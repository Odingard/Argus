# Contributing to ARGUS Core

Thank you for your interest in improving the base mapping logic of the ARGUS Substrate! 

## Setting up your environment

1.  Clone this repository locally.
2.  Run `python -m venv venv` and `source venv/bin/activate`.
3.  Install development dependencies using `pip install -e .`

## Submission Guidelines

*   **Pull Requests**: All PRs should target the `main` branch. 
*   **Agent PRs**: We welcome submissions for new basic heuristic agents (Layers 1 and 2). However, advanced semantic synthesizers, zero-day compilers, and exploit vector engines will strictly not be accepted into the Open-Source Core branch, as they are securely maintained in the ARGUS Enterprise registry.
