# Security Policy

## Supported Versions
Only the latest major release of `argus-core` is officially supported for security updates. 

## Reporting a Vulnerability

As this is a cybersecurity research framework, we take framework-level vulnerabilities seriously (e.g., prompt injection vulnerabilities within our own agent swarm structure).

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, report them directly by sending an email to:
👉 **security@sixsenseenterprise.com**

We will acknowledge receipt of your vulnerability report within 48 hours and strive to send you regular updates about our progress. If the vulnerability poses significant risk to our Enterprise deployments, we may initiate a coordinated disclosure timeline.
